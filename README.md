# Nodejs 
This project is generated using Node Js

## Environment Setup 
Need to install the latest version of nodejs  
```PowerShell
node --version
```


## To create package.json
```PowerShell
npm init -y
```  
```PowerShell
npm i -D nodemon
``` 
```PowerShell
npm i express express-handlebars express-session mongoose passport passport-local
```  
```PowerShell
npm i bcrypt
``` 
## Run
```Terminal
npm start
```
